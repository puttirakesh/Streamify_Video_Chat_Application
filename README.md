# Streamify_Video_Chat_Application
A full‑stack real‑time communication platform combining chat and video calls. It supports text messaging with reactions and typing indicators, plus one‑on‑one and group video calls with screen sharing and recording. Built with React, TailwindCSS, Express.js, MongoDB, and secured by JWT authentication, it’s scalable, secure, and user‑friendly.

## 🧪 .env Setup

### Backend (`/backend`)

```
PORT=5001
MONGO_URI=your_mongo_uri
STEAM_API_KEY=your_steam_api_key
STEAM_API_SECRET=your_steam_api_secret
JWT_SECRET_KEY=your_jwt_secret
NODE_ENV=development
```

### Frontend (`/frontend`)

```
VITE_STREAM_API_KEY=your_stream_api_key
```

---

## 🔧 Run the Backend

```bash
cd backend
npm install
npm run dev
```

## 💻 Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
